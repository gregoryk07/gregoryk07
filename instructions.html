Register Protocol

This command line tool takes 4 arguments. It won't work if there are more or less than that.

  1. First argument takes absolute path to the program you want to use protocol for.
  2. Second argument takes information about how will the protocol look like, for example if you put "protocol" as argument the protocol will be "protocol://"
  3. Third argument takes information about program name.
  4. Fourth argument is simply add or remove. It decides if tool adds or removes an protocol.
